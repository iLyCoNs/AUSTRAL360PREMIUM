# KPK V1 — Plan Premium

Sala de ventas 360° completa con copiloto Jarvis/Gigi.

## Incluye
- Todo lo del Plan Básico
- Copiloto IA + voz TTS/STT
- Agenda de visita, financiamiento, turismo, clima
- Cercanos OSM + mapa + widgets arrastrables
- Admin con sección **Copiloto IA** (APIs, voces, módulos ON/OFF)

## Arranque
Abre `index.html`. Admin: `admin.html` → pestaña Copiloto IA.

Plan flag: `data/plan.json` · Features también en `data/brand.json` → `features`
